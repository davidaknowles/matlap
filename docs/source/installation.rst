Installation
============

Requirements
------------

- Python ≥ 3.10
- JAX ≥ 0.4 (with optional GPU support via CUDA)

Install from source
-------------------

.. code-block:: bash

    git clone https://github.com/davidaknowles/matlap.git
    cd matlap
    pip install -e .

Install with dev extras
-----------------------

.. code-block:: bash

    pip install -e ".[dev]"
